## Funcionamento do site;

Esse site é um sistema funcional que automatiza processo recebe informações guarda, organiza e simplifica para o usuário. Meu objeto é deixar o mais intuitivo possível para facilitar o acesso a automações de planilhas no excel ou formulários;

consiste em receber informações cadastrais de usuários, Armazenar e facilitar a Automação:

* Sendo organizado no NEXT SQL + JS + REACT (Backend)
* E o front end é o HTML + CSS + JS + REACT (Frontend)

A fim de pegar essas informações para que possamos preencher formulários de forma mais prática onde o usuário coloca o link do site ou aplicativo e cadastra de forma automática com (IA) ou manualmente especificando o caminho dos dados tem o objetivo de ser mais prático para ajudar qualquer um a automatizar seus processos e GERENCIAR todos os usuários cadastrados com métricas em power BI e com dashboards produzido de forma automática e com alterações facilitadas;

* criar e apagar objetos (Usuários)
* editar suas informações e adicionar automaticamente

objeto principal é entender o fluxo dos dados Redes funções requisições web, funcionamento do armazenamento dentro do banco de dados ligações diretas com o código escrito e todos os demais processos na criação desse site SPA
